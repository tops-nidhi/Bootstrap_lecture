->Bootstrap provides a series of classes that can be used to apply various styling to the tables

->The .table class is used to create a simple Bootstrap table. This class name is used with the <table> tag to create a table.

-> The .table-stripped class is used to create an alternate dark and light rows. Use the combination of table and table-stripped classes within the <table> tag to create a striped table.

->The .table-bordered class is used to add borders on all sides of the table and cell. Use the combination of table and table-bordered classes within the <table> tag to create bordered table.

->The .table-hover class is used to add the hover effect on table rows. Use the combination of table and table-hover classes within the <table> tag to create a hover rows table.

->The .table-dark class is used to add the black background color of a table. Use the combination of table and table-dark classes within the <table> tag to create a dark table.

->The .table-dark and .table-stripped classes are used to create a dark striped table. Use the combination of table, table-dark, and table-stripped classes within the <table> tag to create the dark striped table.

->The .table-borderless is used to remove the border from the table. Use the combination of table and table-borderless classes within the <table> tag to create a borderless table.

->The .table-sm class is used to create a small table by reducing the cell padding. Use the combination of table, table-bordered, and table-sm classes within the <table> tag to create a bordered small table.
